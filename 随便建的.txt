红色的是重要的。
Git init   在指定的文件夹里已经初始化一个空的仓库。  非常重要，不要删除它是一个隐藏的文件。
cls  清屏
git status   查看文件的状态 
 红色  代表当前文件处于工作区，还没有提交到暂存区
 绿色代表当前文件处于暂存区，还没有提交到历史区。
git add 把指定文件从工作区提交到暂存区。
git rm --cached    删除暂存区的信息，这个操作很少用。
git add -A  /git add .把工作区最新修改的文件 提交到暂存区
git commit -m'注释这是里面写内容'  把暂存区的东西提交到历史区
git log/查看历史版本信息。
git reflog 在log的基础上增加历史回滚的信息。
git reset --hard  回滚  后面写历史版本号的前七位。
 建立连接  git remote add origin 仓库地址  回车
